Тема

Структура HTML-документа. Вибір предметної галузі. Робота з посиланнями, таблицями, зображеннями, списками в HTML-документі.

Мета

Придбати практичні навички роботи з HTML-документом, таблицями, зображеннями, посиланнями, списками, формами. Створити шаблон звітного HTML-документа для відображення результатів роботи всіх лабораторних робіт.

Опис предметної галузі

Сайт тендерів: Веб-додаток надає можливість користувачам переглядати активні тендери, шукати потрібні за допомогою ключових слів, подавати пропозиції, а також створювати власні тендери.

Структура головної сторінки

На головній сторінці розміщено зображення, таблиця тендерів, списки функцій, посилання на інші сторінки та форму для пошуку тендерів.

Зображення

<img src="images/tender.jpg" alt="Тендери" />

Теорія: Тег <img> використовується для вставлення зображень на сторінку. Основні атрибути:

  • src – шлях до зображення.
  • alt – альтернативний текст при відсутності зображення.

Таблиця тендерів

<table>
  <tr><th>Номер тендеру</th><th>Опис</th><th>Стан</th></tr>
  <tr><td>001</td><td>Будівництво дороги</td><td>Активний</td></tr>
  <tr><td>002</td><td>Закупівля обладнання</td><td>Завершений</td></tr>
</table>

Теорія: Теги для створення таблиць:

  • <table> – огортає таблицю.
  • <tr> – рядок таблиці.
  • <th> – заголовок стовпця.
  • <td> – комірка таблиці.
Номер тендеру Опис Стан

002 Закупівля обладнання Завершений

Списки

<ul>
  <li>Перегляд тендерів</li>
  <li>Пошук тендерів</li>
  <li>Подання пропозицій</li>
</ul>

<ol>
  <li>Створення тендеру</li>
  <li>Управління тендером</li>
  <li>Завершення тендеру</li>
</ol>

Теорія: Для створення списків використовуються теги <ul> (ненумерований список) та <ol> (нумерований список). Тег <li> позначає елемент списку.

  • Перегляд тендерів
  • Пошук тендерів
  • Подання пропозицій
  1. Створення тендеру
  2. Управління тендером
  3. Завершення тендеру

Форма пошуку тендерів

<form action="/search" method="get">
  <input type="text" name="query" placeholder="Введіть ключові слова" />
  <button type="submit">Пошук</button>
</form>

Посилання

<a href="about.html">Опис предметної галузі</a>

Теорія: Тег <a> використовується для створення посилань. Основний атрибут – href, який містить адресу сторінки.

Висновки

Під час виконання лабораторної роботи я навчився створювати HTML-документ, використовувати зображення, таблиці, списки та посилання. Також навчився працювати з формами та атрибутами тегів.
